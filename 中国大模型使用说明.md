## 使用指南

### 1. 添加 Provider

1. 打开扩展设置页面
2. 点击「添加 Provider」
3. 选择对应的厂商
4. 填入 API Key
5. 点击「获取模型列表」自动获取可用模型
6. 选择需要使用的模型
7. 保存

### 2. 通义千问 (DashScope) 配置

**获取 API Key:**
1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 创建 API Key

**常用模型:**
- `qwen-plus`
- `qwen-max`
- `qwen-turbo`
- `qwen-long` (长上下文)
- `qwen-vl-max` (视觉)

### 3. 智谱 GLM 配置

**获取 API Key:**
1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
2. 创建 API Key

**常用模型:**
- `glm-4-plus`
- `glm-4-9b`
- `glm-4-air`
- `glm-4v` (视觉)

### 4. DeepSeek 配置

**获取 API Key:**
1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 创建 API Key

**常用模型:**
- `deepseek-chat`
- `deepseek-reasoner`

### 5. 月之暗面 (Kimi) 配置

**获取 API Key:**
1. 访问 [月之暗面平台](https://platform.moonshot.cn/)
2. 创建 API Key

**常用模型:**
- `moonshot-v1-128k`
- `moonshot-v1-32k`

### 6. 腾讯混元配置

**获取 API Key:**
1. 访问 [腾讯云 API 密钥管理](https://console.cloud.tencent.com/cam/capi)
2. 创建 SecretId 和 SecretKey

**常用模型:**
- `hunyuan-lite`
- `hunyuan-standard`
- `hunyuan-pro`

### 7. 讯飞星火配置

**获取 API Key:**
1. 访问 [讯飞开放平台](https://console.xfyun.cn/services/spark)
2. 创建 API Key

**常用模型:**
- `spark-v4`
- `spark-v3.5`

### 8. 文心一言配置

**获取 API Key:**
1. 访问 [百度智能云千帆平台](https://console.bce.baidu.com/qianfan)
2. 创建 API Key

**常用模型:**
- `ernie-latest`
- `ernie-pro`
- `ernie-4.0`

## Anthropic 兼容接口使用

如果某个厂商提供的是 Anthropic 兼容接口，可以使用 `anthropic-compat` provider：

1. 选择「自定义 (Anthropic 兼容)」
2. 填入 Base URL
3. 填入 API Key
4. 支持的认证方式：
   - 标准 Anthropic 格式: `x-api-key` header
   - Bearer token 格式: `Authorization: Bearer xxx`

### Q: 哪些模型支持视觉功能？

A: 通常模型 ID 中包含：
- `-vl`
- `-vision`
- `glm-4v`

### Q: 如何处理 CORS 问题？

A: 扩展在 background 脚本中执行 API 调用，不受浏览器 CORS 限制。

### Q: 支持流式输出吗？

A: 支持！扩展支持流式输出，可以在设置中开启。
