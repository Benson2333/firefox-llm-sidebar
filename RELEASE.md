## 📋 安装说明

### 开发模式安装
1. 下载 `llm_-0.3.0.zip` 并解压
2. 打开 Firefox，访问 `about:debugging`
3. 点击 "此 Firefox"
4. 点击 "加载临时附加组件"
5. 选择解压目录中的 `manifest.json`

### 从源码安装
1. Clone 仓库: `git clone https://github.com/Benson2333/firefox-llm-sidebar.git`
2. 安装依赖: `npm install`
3. 运行: `npm run run`
4. 打包: `npm run build`

## 🎯 使用指南

### 添加中国大模型
1. 打开扩展设置
2. 点击 "添加 Provider"
3. 选择对应的中国厂商
4. 填写 API Key
5. 点击 "获取模型列表"
6. 选择要使用的模型
7. 保存配置

### 使用本地文件
1. 在 Firefox 中打开本地文件
2. 允许扩展访问本地文件
3. 点击侧边栏的 "总结" 或 "翻译"
